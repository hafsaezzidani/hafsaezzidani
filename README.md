##  Featured Projects

### SMS Spam Detection System

Built a complete NLP pipeline for SMS spam classification using real-world imbalanced data.

- Designed the full pipeline: preprocessing, feature engineering, vectorization, modeling, and evaluation
- Compared several models: **Naive Bayes, SVM, and Logistic Regression**
- Implemented **Multinomial Naive Bayes from scratch** to better understand probabilistic classification
- Achieved strong performance on the **UCI SMS Spam Collection** dataset:

```text
F1-score : 94%
Precision: 98%
Recall   : 90%
Dataset  : 5,574 SMS messages, ~13% spam
```

---

### Milk Production Forecasting

Time series forecasting project based on monthly milk production data.

- Analyzed a real monthly production series with **168 observations**
- Applied the **Box-Jenkins methodology**
- Built and evaluated an **ARIMA** forecasting model
- Evaluated forecasts over a 12-month horizon

```text
RMSE : 10.85
MAPE : 1.16%
```
